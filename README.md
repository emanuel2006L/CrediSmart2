Realizamos la integración de nuestro proyecto credismart en react, dándole funcionalidad a la página con funciones como los filtros, las cards dinámicas, las fórmulas y demás características.

Las tecnologías usadas fueron:

- React + Vite
- React Router
- CSS
- HTML
- JS

  Características
- Visualización de las cards
- Navegación entre pantallas
- Envío de ID del crédito seleccionado mediante estado
- Diseño responsivo 
- Página para aplicar al crédito
- Direccionamiendo de botones

- Instalación
  
npm install
npm run dev
